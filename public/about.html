About LootScout

LootScout is a simple tool for finding cheap game deals without the noise. No ads, no accounts, no clutter — just deals.

How it works

We pull deals from CheapShark, a free API that aggregates prices across multiple game stores. Game details and screenshots come from the Steam API.

Why?

Most deal sites are bloated. LootScout strips it back to what matters — price, discount, and a link to buy.

Stack

  • Node.js + Express (backend proxy)
  • Vanilla HTML, CSS, JS (frontend)
  • CheapShark API + Steam API (data)
